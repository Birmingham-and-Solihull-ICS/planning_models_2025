#  BSOL ICB Waiting list projecitons
## Modelling to support multiyear planning round 2025

Aim of this process is to take monthly waiting list figures, per treatment speciality, analyse the waiting list and calculate the required capacity to meet waiting list targets by set dates.
These capacity requirements, in turn, will be added to population based activity plan as additional activity to bring WL into line with targets.
